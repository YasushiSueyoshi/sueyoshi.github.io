A

  • AI ：人工知能（Artificial Intelligence）の総称で、人間の知的活動を模倣・拡張する技術やシステム。
  • AIガバナンス・ゴール ：AI開発・利用において守るべき倫理や社会的責任を示す目標。
  • AIマネジメントシステム ：AIに関する品質やリスクを一元管理する仕組み。国際標準化が進む。
  • AIレベル1 ：単純なプログラム、条件分岐、AI家電
  • AIレベル2 ：ルールベース、チャットボット、音声認識、質疑応答
  • AIレベル3 ：機械学習、自らデータパターンを見出し、最適な出力を戻せる。検索エンジンなど
  • AIレベル4 ：ディープラーニング、自ら特徴量の調整機能がある、自動運転技術
  • AI開発者(AI Developer) ：AIを設計・実装・検証する技術者。
  • AI活用原則：AIを活用する全ての人が守るべき原則として、「人間中心」、「安全性」、「公平性」、「プライバシー保護」、「セキュリティ確保」、「透明性」、「アカウンタビリティ（説明責任）」、「教育・リテラシー」、「公正競争確保」、「イノベーション」の10原則を掲げている。
  • AI効果 ：AIが普及するにつれ、かつてAIと見なされていた技術が当たり前の存在になってしまう現象。
  • AI提供者(AI Provider) ：開発したAI技術や製品を、ユーザーや企業に提供する事業者。
  • AI利用者(AI Business User) ：業務でAIサービスを導入・活用する企業や組織。
  • AIの冬 ：AIへの過剰な期待が失望に変わり、研究開発資金が激減した停滞期。
  • AttentionMechanism ：注意機構全般を指し、入力の重要部分を強調して処理する手法。
  • Attention層 : 力シーケンス内の要素間の関連度を計算し、重要度を再重み付けして特徴を抽出する層。(Transformerモデル関連)
  • 弱いAI(ANI) ：特定のタスクに特化したAI。限られた範囲で高い性能を発揮する。
  • 強いAI(AGI) ：人間レベルの汎用的な知能を持つAI。多様な課題をこなせる。
  • ALBERT(a Lite BERT) ：Google。BERTのパラメータを圧縮し、高速かつ軽量化したモデル。低コストのPCやスマホでも動く。

    • B

  • BERTモデル ：2018.10 Google。双方向の文脈を学習するTransformerベースの言語モデル。MLMとNSPを採用。モデルサイズと学習時間が課題。

    • C

  • ChatGPT ：GPTベースの対話特化型モデル。自然な会話が可能なチャットボット。強化学習を用いて人間のフィードバックを反映。ChatGenerativePre-trainedTransformer
  • Claude ：Anthropic社の大規模言語モデル。安全性や倫理面を重視して開発されている。
  • CNN(畳み込みニューラルネットワーク) ：画像処理などで用いられる、畳み込み演算を特徴とするニューラルネットワーク。
  • Code Interpreter ：ChatGPTなどでコードを実行・解釈し、結果を返す機能。
  • Context ：AIモデルが処理を行う際の参照情報や前後の文脈。

    • F

  • Few-Shotプロンプティング ：少数の例示や補足情報を提示して、AIが期待される出力を学習しやすくする手法。

    • G

  • GAN(敵対的生成ネットワーク) ：画像生成AIで利用。生成器と識別機を対立的に学習させ、リアルなデータを生成する手法。StyleGANはNVIDA社が開発した顔画像の生成特化。
  • Gemini ：Google DeepMindが開発中とされる次世代の大規模言語モデルの名称。
  • GPT-1 ：2018 最初期のGPTモデル。自己回帰型言語モデルの可能性を示した。1億7000万パラメータ
  • GPT-2 ：2019 GPT-1より大幅にパラメータを増やし、文章生成の自然さや多様性を向上させたモデル。15億パラメータ。悪用のリスクあり限定公開
  • GPT-3 :OpenAIが2020年に発表、1750億パラメータを持つ大規模言語モデルで、多様なタスクに対応可能。1750億パラメータ。調整なしでNLPタスクが実行可能
  • GPT-3.5 :2022年、GPT-3の改良版。対話性能や推論能力を強化し、ChatGPTにも使用。3550億パラメータ。RLHFの仕組みを採用
  • GPT-4 ：2023年、OpenAIが開発した最新の大規模言語モデル。高い推論力とマルチモーダル(画像、音声など)対応が特徴。約1兆7600億のパラメータ
  • GPT-4o ：多言語対応かつマルチモーダルなGPTである。
  • GPTs ：GPTsとは、自然言語を入力するだけでオリジナルのチャットボットを作れるサービスです
  • GPTモデル ：2018.6 OpenAI。Transformerベースの自己回帰言語モデル。大規模テキスト学習による生成能力が高い。

    • I

  • Input Data ：モデルに入力するデータ。テキストや画像など様々な形態がある。
  • InstructGPT ：GPT-3をベースに、ユーザーの指示に最適化されたモデル。RLHFの仕組みを採用
  • Instruction ：AIに対して実行してほしい内容を明示的に指示する文面。

    • L

  • LLM ：Large Language Model（大規模言語モデル）の略。大規模データで学習した強力な言語モデル。TemperatureとTop-pのパラメータが設定
  • LM ：Language Model（言語モデル）の略。言語の統計的性質を学習するモデル。
  • LSTM(長・短期記憶) ：RNNの一種で、ゲート機構(forget gate,input gate,output gate)により長期依存関係を学習しやすくしたモデル。1977年に提案
  • Luma Dream Machine ：Luma AIが開発する3D生成やAR向けのAI技術プラットフォーム。

    • M

  • MLM(Masked Language Model) ：一部の単語をマスクし、元の単語を推測させることで文脈理解を学習する手法。

    • N

  • n-gramモデル ：n個連続する単語（文字）の頻度に基づいて文を生成・予測する単純な言語モデル。バイグラム(n=2)、トライグラム(n=3)
  • NSP(Next Sentence Prediction) ：2つの文が連続しているかを学習させ、文章構造理解を深めるタスク。ある文の次に来る可能性のある文を予測。

    • O

  • OpenAI ：GPTシリーズやChatGPTを開発するAI研究機関兼企業。
  • Output Indicator ：モデルの出力に求められる指標や品質基準を示すもの。

    • R

  • RNN ：回帰型ニューラルネットワーク過去の情報を記録しながら(回帰型)新しい入力を処理するという特性がある。
  • RNN(回帰型[リカレント]ニューラルネットワーク) ：時系列データや文章など、シーケンスデータを扱うための再帰構造をもつネットワーク。短い依存関係には優れる
  • RoBERTa ：2019年 FacebookAI。BERTを改良し、大規模データと長時間学習で精度を向上させたモデル。
  • RLHF(Reinforcement Learning from Human Feedback) ：人間の評価を報酬として学習し、モデルの出力を最適化する手法。
  • Runway Gen-3 ：Runwayが提供する生成AIツールの一つ。動画生成・編集機能を備える。

    • S

  • Sora ：OpenAIの動画生成AIツール

    • T

  • Temperature ：生成モデルの出力における多様性(ランダム性)を制御するパラメータ。高いほど多様性が増す。
  • Top-p ：生成モデルで上位の確率質量pに含まれる単語のみをサンプリングする手法（Nucleus Sampling）。次に出力する単語を選択する際の手法
  • Transformerモデル ：CNN,RNNを利用しないモデルとして模索、自己注意機構(Self-Attention)を中心とした高性能な深層学習モデル。

    • V

  • VAE(変文自己符号化器) ：画像生成AIで利用。入力データを潜在空間にマッピングし、生成も可能な確率的オートエンコーダ(統計的アプローチ)。連続的で類似したデータの構造を保持できる。

    • Z

  • Zero-Shotプロンプティング ：例示や追加説明なしで、直接タスクを指示してAIに応答させる手法。

あ

  • アーキテクチャ ：モデルやシステムの全体的な設計構造。
  • アカウンタビリティ ：行為や決定の根拠を示し、説明・責任を負うこと。
  • アカウンタビリティ ：自らの行動や結果について説明責任を果たすこと。
  • アライメント(Alignment) ：AIの目的や行動が人間の価値観・倫理観に合致するように調整すること。
  • 安全性 ：システムやサービスが人々に害を与えないよう、十分に保護された状態。
  • 安全性・公平性 ：危険や不利益がなく、公平な扱いが行われることを重視する価値観。
  • アンチウイルスソフトウェア ：マルウェアを検知・駆除するためのソフトウェア。

    • い

  • イノベーション ：新たな発想や技術を取り入れ、社会Alignmentや経済に変革をもたらすこと。。AI基本原則(7つ)⑦
  • インターネットリテラシー ：ネット上の情報を正しく扱い、危険や詐欺を見極めるための知識・技能。
  • 意匠権 ：物品の形状や模様、色彩などのデザインを保護する権利。出願の日から最長25年
  • 位置エンコーディング ：Transformerなどで順序情報を学習させるために入力に付与するベクトル。各単語の位置を数値化。

    • う

  • ヴァーナー・ヴィンジ ：シンギュラリティ概念を広めたSF作家・数学者。
  • ヴィッシング ：電話や音声通話を利用したフィッシング詐欺。Vishing

    • え

  • 営業秘密 ：企業が秘密として管理する有用な情報。不正競争防止法で保護される。
  • エンコーダー : 入力データを潜在表現に圧縮するニューラルネットワーク構成要素。
  • エキスパートシステム ：専門家の知識やルールをプログラム化し、専門家に近い推論を行うシステム。

    • お

  • 重み ：ニューラルネットワークで入力に対して掛け合わされるパラメータ。学習によって更新される。重要性に差異を設ける

か

  • 隠れ層 ：入力層と出力層の間にある層。中間表現を学習する役割を担う。モデルの表現力向上、過剰だと過学習に繋がる。
  • 過学習 ：AIが与えられたデータに適応し過ぎてしまい、未知のデータに対する処理精度が落ちる現象
  • 画像のリサイズ ：画像のピクセル数を変更する処理。分析や学習でサイズを揃えるために行う。
  • 改正個人情報保護法 ：個人情報保護法を改正し、国際的なデータ取扱いなどルールを強化した法。
  • 学習済みモデル ：すでにデータを使って学習が行われ、推論や予測に利用できる状態のモデル。
  • 環境・リスク分析 ：AI活用に伴う社会的・技術的リスクを評価し、対策を検討するプロセス。

    • き

  • 機械学習 ：データからパターンを学習し、予測や分類などを自動化する手法の総称。
  • 機微(センシティブ)情報 ：差別や偏見に繋がる恐れがある、特に慎重な保護が必要な情報。
  • 偽情報(ディスインフォメーション) ：意図的に流布される虚偽や誤解を招く情報。
  • 強化学習 ：報酬という特定の目的を設定し、報酬を最大化する行動を学習する枠組み。試行錯誤を通じて最適な方策を獲得する。
  • 教育・リテラシー ：AIや情報技術を正しく扱うために必要な知識・技能を習得すること。。AI基本原則(7つ)②
  • 教師あり学習 ：正解ラベル付きのデータを用いて学習を行い、未知の入力に対してラベルを予測する手法。利用例・・・音声性のトレーニングに用いられる。
  • 教師なし学習 ：ラベルのないデータから、その構造やパターンを発見する学習手法。

    • く

  • クラスタリング ：教師なし学習の一種で、類似度の高いデータ同士をグループ化する手法。データの理解や分析、異常検知などに活用。

    • け

  • 限定提供データ ：提供先や用途が制限されたデータ。契約などで利用範囲を厳格に設定。限定提供性、相当量備蓄性、電磁的管理性。

    • こ

  • 個人識別番号 ：個人を一意に特定する番号（例: マイナンバー）。
  • 個人情報取扱業者 ：個人情報(データベース等)を事業で扱う者。適切な管理が法律で義務付けられる。
  • 個人情報保護委員会 ：個人情報保護法の運用や監督を行う日本の行政機関。2016.1.1に設立。独立性が高い
  • 個人情報保護法 ：個人情報の適正な管理・利用を目的とした日本の法律。
  • 公正競争確保 ：事業者間の不正行為や独占的な活動を防ぎ、消費者や社会にとって公平・健全な市場環境を維持するための取り組みや制度。。AI基本原則(7つ)⑤
  • 公平性 ：不当な差別や偏りがないよう、平等に扱う原則。⑥公平性・説明責任

し

  • 識別機 : GANで入力が本物か偽物かを判定するネットワーク。
  • 次元削減 ：教師なし学習の一種で高次元データをより低次元へ圧縮し、情報の要点を抽出する手法。
  • 自己回帰モデル ：1990-2000年　過去の値を用いて将来の値を予測する。GPTなどのモデルで利用（Autoregressive Model）。
  • 自己注意力(Self-Attention) ：要素同士が相互に注意を向ける機構。入力をQuery,Key,Valueの行列から各要素が他の要素とどれだけ関連性があるかを計算し、注意力スコアを計算。
  • シーケンスデータ ：連続性を持つデータ（テキスト・音声・時系列など）。
  • シナプス ：ニューロン同士を接続する部分。ニューラルネットワークにおける重みに相当する。
  • シンギュラリティ(技術的特異点) ：AIが人間の知能を超える理論上の転換点。社会に大きな変化をもたらすとされる。
  • 持続可能な社会(Sustainability) ：現在と将来の世代が共存し、環境・経済・社会のバランスを保ちながら発展する社会。
  • 商標権 ：商品やサービスの区別に用いられる商標の独占使用権。設定登録から10年
  • 肖像権 ：個人の顔や姿を無断で撮影・利用されないよう保護する権利。
  • 情報の重みづけ ：ニューラルネットワークで入力特徴に対して重要度を調整する仕組み。
  • 情報リテラシー ：情報を収集・評価・活用する能力。
  • 人工ニューロン(ノード) ：生物学的ニューロンを模倣した演算ユニットで、複数の入力を重み付けして出力する。

    • す

  • 推論 ：規則や知識に基づき、新たな結論や知見を導き出すプロセス。
  • スピアフィッシング ：特定の個人や組織を狙ったフィッシング詐欺。より巧妙で標的に合わせた手口。
  • スミッシング ：SMSを使ったフィッシング詐欺。URLをクリックさせ情報を盗む。Smishing

    • せ

  • 生成器 : GANで偽データを生成するネットワーク。識別機を騙せるように学習する。
  • 制限付きボツマンマシン ：1986年　可視層と隠れ層の間にのみ結合をもつ簡易型ボルツマンマシン。データの生成や再構成に利用。
  • 正規化 ：データを一定のスケールに揃え、学習の安定性を高める手法。L
  • 正則化 ：過学習を防ぐために、パラメータを調整・制限して、モデルの複雑さを抑える手法（li・L2正則化など）。L2正則化はモデルのパラメータ値を0に近づける手法
  • 生成AI(ジェネレーティブAI) ：テキストや画像、音声、動画など様々なコンテンツを生成する能力を持つAI。
  • セキュリティとプライバシー ：情報やシステムの安全性を確保し、個人や組織のプライバシーを保護する概念。
  • セキュリティ確保 ：情報やシステムを外部の脅威から守る対策。。AI基本原則(7つ)③プライバシーとセキュリティ
  • セキュリティ確保 ：情報資産を不正アクセスや漏洩から守るための対策を講じること。。AI基本原則(7つ)④
  • 潜在べクトル ：データを圧縮した内在的な表現ベクトル。生成モデルなどで利用される。

    • そ

  • ソーシャルエンジニアリング攻撃 ：人間の心理的な隙をつき、情報や権限を不正に取得する攻撃手法。

た

  • ダートマス会議 ：1956年に開催され、AIという研究分野の名称と概念が正式に提案された会議。
  • 第一次AIブーム ：1950年代後半から1960年代にかけてのAI研究隆盛期。探索や推論に期待が集まった。1956ダートマス会議～
  • 第三次AIブーム ：2000年代後半～ ビッグデータやディープラーニングでAIが再興した時期。
  • 第二次AIブーム ：1980年代にエキスパートシステムを中心にAI研究・開発が活発化した時期。
  • 畳み込み ：入力データの局所領域から特徴を抽出する演算処理。
  • 多様な背景を持つ人々が多様な幸せを追及できる社会(Diversity&Inclusion) ：人種・性別・年齢などの多様性を受け入れ、誰もが幸福を追求できる社会。
  • 探索 ：問題空間を系統的に調べ、最適解や有望な解を見つけるプロセス。

    • ち

  • 知的財産権 ：創作物や発明を保護する法律上の権利の総称。
  • 著作権 ：文学や音楽などの著作物を創作した者に与えられる権利。
  • 著作権侵害 ：無断で著作物を複製・公開・頒布するなど、著作権者の権利を侵害する行為。

    • つ

  • 強いAI(AGI) ：人間レベルの汎用的な知能を持つAI。多様な課題をこなせる。

    • て

  • ディープフェイク(深層偽造)技術 ：AIを使って人物の動画や音声を本物のように加工・合成する技術。
  • ディープラーニング ：多層のニューラルネットワークを用いた機械学習手法の総称。
  • データセット ：学習や評価に用いるデータの集合。教師データ・テストデータなどに分割される。
  • データの水増し(augmentation) ：データを回転・反転などで増やし、汎化性能を向上させる手法。
  • データ拡張技術 ：学習用データを多様化するための手法全般。画像・文章・音声などで利用される。非現実的な変換を避ける。
  • テクノロジーの理解 ：科学技術の原理や活用方法を正しく把握すること。
  • デコーダ : 潜在表現から元の形式に近い出力を再構成するニューラルネットワーク構成要素
  • デジタル市民権 ：インターネット社会での権利と責任を理解し、適切に行動すること。
  • 転移学習 ：あるタスクで得た学習済みモデルや知識を別のタスクに応用する手法。

    • と

  • 透明性 ：システムや意思決定プロセスを開示し、公正性や信頼を担保すること。
  • 透明性 ：内部の情報やプロセスを必要に応じて開示し、理解や監督を可能にする。
  • 匿名加工情報 ：個人が特定できないように加工されたデータ。第三者に提供する場合、個人情報保護法により、あらかじめその旨を公開する必要あり。
  • 特許権 ：新規発明に対して、一定期間独占的に実施できる権利。出願から20年
  • 特徴量 ：モデルが学習や推論に用いる、入力データの重要な指標やパターン。
  • ドロップアウト ：学習時にランダムにニューロンを無効化し、過学習を防ぐ手法。

に

  • 2045年問題 ：レイ・カートワイルが提唱した、2045年にAIが人間を超える可能性があるとする議論。
  • ニューラルネットワーク ：多数の人口ニューロンを多層に結合したモデルで、複雑なパターン学習が可能。
  • ニューラル言語モデル ：人工ニューラルネットワークを用いた言語モデル。文章生成、翻訳、質疑応答など。RNNやTransformerなどが代表。
  • ニューロン ：脳神経細胞。ニューラルネットワークの基礎的概念のもととなる単位。活性化関数を用いて入力データを変換。
  • 人間の尊厳が尊重される社会(Dignity) ：すべての人が固有の尊厳を持ち、その価値が認められる社会。
  • 人間中心 ：AIやテクノロジー開発で、人間の尊厳や安全性を中心に設計すること。
  • 人間中心の考え方 ：技術や制度を設計・運用する際、人間の価値や幸福を最優先に考慮する理念。AI基本原則(7つ)①人間中心

    • の

  • ノイズ ：データに含まれる不要な変動や誤差。適切な量なら汎化性能を高める場合もある。
  • ノーフリーランチ定理 ：どのアルゴリズムも、あらゆる問題設定において常に最良とは限らないという定理。

は

  • ハイパーパラメータ ：学習率やバッチサイズ、隠れ層の数など、学習過程で手動設定が必要なパラメータ。
  • パブリシティ権 ：有名人などの氏名や肖像から生じる経済的利益を保護する権利。
  • パラメータ ：モデルが学習を通じて調整する数値（重みやバイアスなど）。
  • ハルシネーション(Hallucination) :AIが事実に基づかない情報をあたかも真実のように生成する現象。
  • 半教師あり学習 ：わずかなラベル付きデータと大量のラベルなし(未知)データを併用して学習する手法。精度は教師あり学習より低い。

    • ひ

  • ビックデータ ：従来の技術では扱いきれないほど大量・多様・高速なデータの総称。

    • ふ

  • ファインチューニング ：事前学習済みモデルを特定のタスクに適応させること。
  • フィッシング詐欺 ：偽のメールやサイトで個人情報を騙し取る行為。
  • 不正競争防止法 ：営業秘密の保護や他社の信用を傷つける行為を規制し、公正な競争を確保する法律。
  • プライバシー保護 ：個人の情報や生活領域を尊重し、不正な利用や侵害を防止する。。AI基本原則(7つ)③プライバシーとセキュリティ
  • ブラックメール ：脅迫的な内容で金銭や情報を要求する不正行為。
  • プレテキスト ：架空の身分や状況を使い、ターゲットを信用させ情報を引き出す手口。ソーシャルエンジニアリングの一種
  • プレトレーニング ：大量のデータで事前学習を行うことで、下流タスクに応用しやすくする手法。一般的な知識や言語のパターンを理解。特定タスク用にファインチューニングされてない。
  • プロンプト ：AIに与える指示文やコンテキスト。出力の内容に大きな影響を与える。
  • プロンプトエンジニアリング ：AIに効果的な指示を与えるための設計・テクニック。

    • へ

  • ベイト攻撃 ：ユーザーが興味を持ちそうな「餌」（ベイト）を用意し、クリックやダウンロード、あるいは物理的な接触を誘導。

    • ほ

  • ボルツマンマシン ：1980年代後半　確率的なエネルギーモデルを用いるニューラルネットワークの一種。確立的に振る舞うユニット間の相互結合型ニューラルネットワーク。学習に時間かかりずきた

ま

  • マスキング ：個人情報の一部を伏せ字や置換することで、匿名性を高める技法。
  • マルウェア ：コンピュータに有害な動作をするソフトウェア。ウイルスやワームなどを含む。
  • マルチモーダル ：テキスト・画像・音声など複数のデータ形式を組み合わせて処理できるAI。

    • め

  • 名誉棄損 ：虚偽の情報などで他者の社会的評価を低下させる行為。

よ

  • 用配慮個人情報 ：人種や病歴など、取り扱いに配慮が必要とされる個人情報。
  • 弱いAI(ANI) ：特定のタスクに特化したAI。限られた範囲で高い性能を発揮する。

ら

  • ランサムウェア ：ファイルを暗号化し、身代金を要求するマルウェア。

    • り

  • リカント層 ：RNNの再帰的な構造を形成する層（リカレント層）。過去の状態を考慮する。音声のリアルタイム処理などで活用
  • リマスタリング ：元データを高解像度化や画質向上する再処理。映像・音声などで用いられる。

    • る

  • ルールベース ：事前に定義されたルール（条件と処理）に基づいて推論・判断を行うAI技術。

    • れ

  • レイ・カートワイル ：2045年問題などの予測を行った未来学者で、シンギュラリティを提唱。
