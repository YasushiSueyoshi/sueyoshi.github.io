生成AIパスポート用語集 索引（あいうえお順）

あ

  • AI : 人工知能（Artificial Intelligence）の総称で、人間の知的活動を模倣・拡張する技術やシステム。
  • AIガバナンス・ゴール : AI開発・利用において守るべき倫理や社会的責任を示す目標。
  • AI開発者(AI Developer) : AIを設計・実装・検証する技術者。
  • AI効果 : AIが普及するにつれ、かつてAIと見なされていた技術が当たり前の存在になってしまう現象。
  • AI提供者(AI Provider) : 開発したAI技術や製品を、ユーザーや企業に提供する事業者。
  • AI利用者(AI Business User) : 業務でAIサービスを導入・活用する企業や組織。
  • AIマネジメントシステム : AIに関する品質やリスクを一元管理する仕組み。国際標準化が進む。
  • AIレベル1 : 単純なプログラム、条件分岐、AI家電
  • AIレベル2 : ルールベース、チャットボット、音声認識、質疑応答
  • AIレベル3 : 機械学習、自らデータパターンを見出し、最適な出力を戻せる。検索エンジンなど
  • AIレベル4 : ディープラーニング、自ら特徴量の調整機能がある、自動運転技術
  • アカウンタビリティ : 自らの行動や結果について説明責任を果たすこと。
  • アーキテクチャ : モデルやシステムの全体的な設計構造。
  • アンチウイルスソフトウェア : マルウェアを検知・駆除するためのソフトウェア。
  • アライメント(Alignment) : AIの目的や行動が人間の価値観・倫理観に合致するように調整すること。
  • ALBERT(a Lite BERT) : Google。BERTのパラメータを圧縮し、高速かつ軽量化したモデル。低コストのPCやスマホでも動く。
  • Attention層 : 入力シーケンス内の要素間の関連度を計算し、重要度を再重み付けして特徴を抽出する層。(Transformerモデル関連)
  • AttentionMechanism : 注意機構全般を指し、入力の重要部分を強調して処理する手法。

い

  • 意匠権 : 物品の形状や模様、色彩などのデザインを保護する権利。出願の日から最長25年
  • InstructGPT : GPT-3をベースに、ユーザーの指示に最適化されたモデル。RLHFの仕組みを採用
  • Instruction : AIに対して実行してほしい内容を明示的に指示する文面。
  • イノベーション : 新たな発想や技術を取り入れ、社会Alignmentや経済に変革をもたらすこと。AI基本原則(7つ)⑦

え

  • 営業秘密 : 企業が秘密として管理する有用な情報。不正競争防止法で保護される。
  • エキスパートシステム : 専門家の知識やルールをプログラム化し、専門家に近い推論を行うシステム。
  • エンコーダー : 入力データを潜在表現に圧縮するニューラルネットワーク構成要素。

お

  • OpenAI : GPTシリーズやChatGPTを開発するAI研究機関兼企業。
  • Output Indicator : モデルの出力に求められる指標や品質基準を示すもの。

か

  • 改正個人情報保護法 : 個人情報保護法を改正し、国際的なデータ取扱いなどルールを強化した法。
  • 過学習 : AIが与えられたデータに適応し過ぎてしまい、未知のデータに対する処理精度が落ちる現象
  • 学習済みモデル : すでにデータを使って学習が行われ、推論や予測に利用できる状態のモデル。
  • 機械学習 : データからパターンを学習し、予測や分類などを自動化する手法の総称。
  • 機微(センシティブ)情報 : 差別や偏見に繋がる恐れがある、特に慎重な保護が必要な情報。
  • 教師あり学習 : 正解ラベル付きのデータを用いて学習を行い、未知の入力に対してラベルを予測する手法。利用例・・・音声性のトレーニングに用いられる。
  • 教師なし学習 : ラベルのないデータから、その構造やパターンを発見する学習手法。
  • クラスタリング : 教師なし学習の一種で、類似度の高いデータ同士をグループ化する手法。データの理解や分析、異常検知などに活用。
  • Claude : Anthropic社の大規模言語モデル。安全性や倫理面を重視して開発されている。
  • 強化学習 : 報酬という特定の目的を設定し、報酬を最大化する行動を学習する枠組み。試行錯誤を通じて最適な方策を獲得する。
  • 強いAI(AGI) : 人間レベルの汎用的な知能を持つAI。多様な課題をこなせる。
  • Context : AIモデルが処理を行う際の参照情報や前後の文脈。
  • GAN(敵対的生成ネットワーク) : 画像生成AIで利用。生成器と識別機を対立的に学習させ、リアルなデータを生成する手法。StyleGANはNVIDA社が開発した顔画像の生成特化。
  • Gemini : Google DeepMindが開発中とされる次世代の大規模言語モデル。
  • GPT-3 : OpenAIが2020年に発表、1750億パラメータを持つ大規模言語モデル。多様なタスクに対応可能。調整なしでNLPタスクが実行可能
  • GPT-4o : 多言語対応かつマルチモーダルなGPTである。
  • GPTs : 自然言語を入力するだけでオリジナルのチャットボットを作れるサービス。

さ

  • シーケンスデータ : 連続性を持つデータ（テキスト・音声・時系列など）。
  • シナプス : ニューロン同士を接続する部分。ニューラルネットワークにおける重みに相当する。
  • 自己回帰モデル : 1990-2000年　過去の値を用いて将来の値を予測する。GPTなどのモデルで利用（Autoregressive Model）。
  • 自己注意力(Self-Attention) : 要素同士が相互に注意を向ける機構。入力をQuery,Key,Valueの行列から各要素が他の要素とどれだけ関連性があるかを計算し、注意力スコアを計算。
  • システム : 複数の要素が相互に作用し合って1つの機能を果たす仕組み全体。
  • 自然言語処理(NLP) : 人間の言語をコンピュータで扱う技術全般。理解・生成などを含む。テキスト生成AIで利用
  • 識別機 : GANで入力が本物か偽物かを判定するネットワーク。
  • 次元削減 : 教師なし学習の一種で高次元データをより低次元へ圧縮し、情報の要点を抽出する手法。
  • 人工ニューロン(ノード) : 生物学的ニューロンを模倣した演算ユニットで、複数の入力を重み付けして出力する。
  • 重み : ニューラルネットワークで入力に対して掛け合わされるパラメータ。学習によって更新される。
  • 商標権 : 商品やサービスの区別に用いられる商標の独占使用権。設定登録から10年
  • 情報の重みづけ : ニューラルネットワークで入力特徴に対して重要度を調整する仕組み。
  • 生成AI(ジェネレーティブAI) : テキストや画像、音声、動画など様々なコンテンツを生成する能力を持つAI。
  • 生成器 : GANで偽データを生成するネットワーク。識別機を騙せるように学習する。
  • Sora : OpenAIの動画生成AIツール
  • ソーシャルエンジニアリング攻撃 : 人間の心理的な隙をつき、情報や権限を不正に取得する攻撃手法。

た

  • ダートマス会議 : 1956年に開催され、AIという研究分野の名称と概念が正式に提案された会議。
  • 第一次AIブーム : 1950年代後半から1960年代にかけてのAI研究隆盛期。探索や推論に期待が集まった。1956ダートマス会議～
  • 第二次AIブーム : 1980年代にエキスパートシステムを中心にAI研究・開発が活発化した時期。
  • 第三次AIブーム : 2000年代後半～ ビッグデータやディープラーニングでAIが再興した時期。
  • ディープラーニング : 多層のニューラルネットワークを用いた機械学習手法の総称。
  • データセット : 学習や評価に用いるデータの集合。教師データ・テストデータなどに分割される。
  • データ拡張技術 : 学習用データを多様化するための手法全般。画像・文章・音声などで利用される。非現実的な変換を避ける。
  • データの水増し(augmentation) : データを回転・反転などで増やし、汎化性能を向上させる手法。
  • テクノロジーの理解 : 科学技術の原理や活用方法を正しく把握すること。
  • デコーダ : 潜在表現から元の形式に近い出力を再構成するニューラルネットワーク構成要素。
  • 特徴量 : モデルが学習や推論に用いる、入力データの重要な指標やパターン。
  • 特許権 : 新規発明に対して、一定期間独占的に実施できる権利。出願から20年
  • ドロップアウト : 学習時にランダムにニューロンを無効化し、過学習を防ぐ手法。
  • Transformerモデル : CNN,RNNを利用しないモデルとして模索、自己注意機構(Self-Attention)を中心とした高性能な深層学習モデル。

な

  • 名誉棄損 : 虚偽の情報などで他者の社会的評価を低下させる行為。
  • ニューラル言語モデル : 人工ニューラルネットワークを用いた言語モデル。文章生成、翻訳、質疑応答など。RNNやTransformerなどが代表。
  • ニューラルネットワーク : 多数の人口ニューロンを多層に結合したモデルで、複雑なパターン学習が可能。
  • ニューロン : 脳神経細胞。ニューラルネットワークの基礎的概念のもととなる単位。活性化関数を用いて入力データを変換。
  • n-gramモデル : n個連続する単語（文字）の頻度に基づいて文を生成・予測する単純な言語モデル。
  • ノイズ : データに含まれる不要な変動や誤差。適切な量なら汎化性能を高める場合もある。
  • ノーフリーランチ定理 : どのアルゴリズムも、あらゆる問題設定において常に最良とは限らないという定理。
  • NSP(Next Sentence Prediction) : 2つの文が連続しているかを学習させ、文章構造理解を深めるタスク。

は

  • バイアス : データや判断における偏り。公平性に影響を与える要因。
  • パブリシティ権 : 有名人などの氏名や肖像から生じる経済的利益を保護する権利。
  • パラメータ : モデルが学習を通じて調整する数値（重みやバイアスなど）。
  • 半教師あり学習 : わずかなラベル付きデータと大量のラベルなし(未知)データを併用して学習する手法。
  • ビックデータ : 従来の技術では扱いきれないほど大量・多様・高速なデータの総称。
  • 弱いAI(ANI) : 特定のタスクに特化したAI。限られた範囲で高い性能を発揮する。
  • 不正競争防止法 : 営業秘密の保護や他社の信用を傷つける行為を規制し、公正な競争を確保する法律。
  • プレトレーニング : 大量のデータで事前学習を行うことで、下流タスクに応用しやすくする手法。
  • プロンプト : AIに与える指示文やコンテキスト。出力の内容に大きな影響を与える。
  • プロンプトエンジニアリング : AIに効果的な指示を与えるための設計・テクニック。
  • ベイト攻撃 : ユーザーが興味を持ちそうな「餌」（ベイト）を用意し、クリックやダウンロード、あるいは物理的な接触を誘導。
  • 変分自己符号化器(VAE) : 画像生成AIで利用。入力データを潜在空間にマッピングし、生成も可能な確率的オートエンコーダ。

ま

  • マスキング : 個人情報の一部を伏せ字や置換することで、匿名性を高める技法。
  • マルウェア : コンピュータに有害な動作をするソフトウェア。ウイルスやワームなどを含む。
  • マルチモーダル : テキスト・画像・音声など複数のデータ形式を組み合わせて処理できるAI。
  • MLM(Masked Language Model) : 一部の単語をマスクし、元の単語を推測させることで文脈理解を学習する手法。

や

  • 有配慮個人情報 : 人種や病歴など、取り扱いに配慮が必要とされる個人情報。

ら

  • ランサムウェア : ファイルを暗号化し、身代金を要求するマルウェア。
  • リカント層 : RNNの再帰的な構造を形成する層（リカレント層）。過去の状態を考慮する。音声のリアルタイム処理などで活用
  • リマスタリング : 元データを高解像度化や画質向上する再処理。映像・音声などで用いられる。
  • ルールベース : 事前に定義されたルール（条件と処理）に基づいて推論・判断を行うAI技術。
  • Luma Dream Machine : Luma AIが開発する3D生成やAR向けのAI技術プラットフォーム。
  • レイ・カートワイル : 2045年問題などの予測を行った未来学者で、シンギュラリティを提唱。
  • RoBERTa : 2019年 FacebookAI。BERTを改良し、大規模データと長時間学習で精度を向上させたモデル。
  • RNN(回帰型[リカレント]ニューラルネットワーク) : 時系列データや文章など、シーケンスデータを扱うための再帰構造をもつネットワーク。
  • RLHF(Reinforcement Learning from Human Feedback) : 人間の評価を報酬として学習し、モデルの出力を最適化する手法。
  • Runway Gen-3 : Runwayが提供する生成AIツールの一つ。動画生成・編集機能を備える。

わ

  • ワーム : コンピュータに有害な動作をするソフトウェアの一種。
